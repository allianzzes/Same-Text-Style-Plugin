# Same Style Select Plugin

Same Text Style Select Plugin lets designers select texts across the design files based on specific typographic properties. This tool is helpful for managing and organizing legacy design files, and files with inconsistent text styles.

## Steps to Use:

1. Upon the launch of the plugin, select a text layer. This text layer will be the point of reference in selecting other text layers. The plugin extracts the typographic properties and lists them.
2. The corresponding checkbox of text properties represents the inclusion during search and selection. 
3. Define the specific scope of search through the search area dropdown. 
4. Click "Select Now". The plugin then selects the text objects with properties matching with the source text layer.

## Tips:

 If no matches were found, try unchecking one property to broaded search.
